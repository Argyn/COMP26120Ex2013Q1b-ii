The format of the input

<strong>NODE</strong> [index] [cost of staying]
<br/>
<strong>EDGE</strong> [index of vertex1] [index of vertex2] [distance]
<br/>
Graph is undirected so EDGE 1 2 will suffice, no EDGE 2 1 is needed.
